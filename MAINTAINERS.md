# Maintainers

This document lists the maintainers of the TomTom Maps SDK for JavaScript.

## Current Maintainers

- Daniel Fornies Soria (@DanielForniessoria-TomTom)
- Suleyman Ali (@SuleymanAli-TomTom)
- Álvaro Graça (@AlvaroGraca-TomTom)

Maintainers are responsible for:
- Triaging issues
- Maintaining code quality and project standards
- Making architectural decisions
- Managing releases

## Contact

For questions or feedback about the project, please:
- Open an [issue](https://github.com/tomtom-international/maps-sdk-js/issues) for bug reports or feature requests
- Start a [discussion](https://github.com/tomtom-international/maps-sdk-js/discussions) for questions and ideas
- Check the [documentation](https://docs.tomtom.com/maps-sdk-js/)

>> [!IMPORTANT]
> We are not currently accepting pull requests. However, we greatly value feedback from the community through issues and discussions.

## Becoming a Maintainer

Due to the proprietary nature of this SDK (see [LICENSE.txt](./LICENSE.txt)), maintainer status is managed internally by TomTom. Check out our open positions at [TomTom Careers](https://www.tomtom.com/careers/).
