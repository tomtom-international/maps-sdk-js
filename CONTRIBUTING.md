## Building from Source

If you want to build the SDK from source for learning or testing:

```bash
git clone https://github.com/tomtom-international/maps-sdk-js.git
cd maps-sdk-js
pnpm install
pnpm build
```

For detailed setup and build instructions, see [documentation/development/](./documentation/development/).
