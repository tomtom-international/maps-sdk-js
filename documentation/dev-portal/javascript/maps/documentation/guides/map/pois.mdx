# Configuring POIs

## Changing POIs visibility
The POI layer visibility can be controlled during runtime with the [`POIsModule`](/javascript/maps/documentation/api-reference/map_POIs.POIsModule) module.

```javascript
import { POIsModule } from "@anw/maps-sdk-js/map";
const pois = await POIsModule.get(map);
pois.setVisible(false);
```

## Filtering POI categories
It's also possible to filter shown POIs by category or classification group using the `filterCategories()` method on [`POIsModule`](/javascript/maps/documentation/api-reference/map_POIs.POIsModule).
See the [Map POI filters](http://hosted-examples.maps-sdk-js.tomtom.com/main/map-poi-filters) example for this functionality.
