# Polygons

The Map Display module supports drawing polygons on the map through the [`GeometryModule`](/javascript/maps/documentation/api-reference/classes/map_geometry.GeometryModule) class.
This module takes a [`GeometryDataResponse`](/javascript/maps/documentation/api-reference/modules/core#GeometryDataResponse) object and draws the geometry on the map.

> A [`GeometryDataResponse`](/javascript/maps/documentation/api-reference/modules/core#GeometryDataResponse) is returned by the Geometry Data service.
Read more about this service [here](/javascript/maps/documentation/guides/search/specialized-search#_geometry_data_search).

> Make sure you have an initialized [`TomTomMap`](/javascript/maps/documentation/api-reference/classes/map_init.TomTomMap) instance in your project.
Read more about initializing the map in the [Quickstart](/javascript/maps/documentation/guides/map-display/quickstart) guide.

```javascript
import { GeometryModule } from "@anw/maps-sdk-js/map";
import { geocode, geometryData } from "@anw/maps-sdk-js/services";

const areaToSearch = await geocode({ query: "paris", limit: 1 });
const areaGeometry = await geometryData({ geometries: areaToSearch });
const geometryModule = await GeometryModule.init(map);
geometryModule.show(areaGeometry);
```

See [Map Geometry Search Playground](http://hosted-examples.maps-sdk-js.tomtom.com/main/map-geometry-search-playground) for a working example.
