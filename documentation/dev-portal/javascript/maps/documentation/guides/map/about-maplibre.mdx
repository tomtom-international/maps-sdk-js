# About MapLibre Integration

This guide explains how the TomTom Maps SDK for JavaScript integrates with MapLibre GL JS and when to use each library directly.

## MapLibre GL JS Overview

[MapLibre GL JS](https://maplibre.org/maplibre-gl-js-docs/api/) is an open-source TypeScript library for publishing interactive maps on websites. It provides powerful capabilities for displaying beautiful maps, adding data, and creating user interactions.

**Key MapLibre Features:**
- High-performance vector map rendering
- Extensive customization options
- Rich interaction capabilities
- Open-source and community-driven
- Cross-platform compatibility

## TomTom's MapLibre Strategy

TomTom officially supports MapLibre as part of our commitment to open-source mapping technologies and OpenStreetMap data integration. The TomTom Maps SDK builds on MapLibre to provide:

- **TomTom-specific functionality**: Seamless integration with TomTom services
- **Simplified APIs**: Easier access to complex mapping features
- **Consistent styling**: Pre-built TomTom map styles and themes
- **Service integration**: Direct connection to TomTom's routing, search, and traffic services

## SDK Architecture and MapLibre Integration

### Shared Dependency Model
The TomTom SDK uses MapLibre as a peer dependency, meaning both libraries coexist in your application:

```json
// package.json dependencies
{
  "dependencies": {
    "@cet/maps-sdk-js": "^0.9.3",
    "maplibre-gl": "^5.7.1"
  }
}
```

### Direct MapLibre Access
Every TomTom map instance provides direct access to the underlying MapLibre map:

```javascript
import { TomTomMap } from '@cet/maps-sdk-js/map';

const tomtomMap = new TomTomMap({ container: 'map' });

// Access the underlying MapLibre map instance
const maplibreMap = tomtomMap.mapLibreMap;

// Use any MapLibre functionality
maplibreMap.on('click', (event) => {
    console.log('Map clicked at:', event.lngLat);
});

// Add MapLibre controls
maplibreMap.addControl(new maplibregl.NavigationControl());
```

## When to Use TomTom SDK vs Direct MapLibre

### Use TomTom SDK For:

**TomTom Service Integration:**
```javascript
// TomTom SDK - Integrated search and display
import { search } from '@cet/maps-sdk-js/services';
import { PlacesModule } from '@cet/maps-sdk-js/map';

const results = await search({ query: 'restaurants', center: [4.9, 52.3] });
const placesModule = await PlacesModule.init(map);
placesModule.show(results);
```

**Simplified Complex Operations:**
```javascript
// TomTom SDK - Easy routing
import { RoutingModule } from '@cet/maps-sdk-js/map';

const routing = await RoutingModule.get(map);
await routing.displayRoute({
    waypoints: [origin, destination]
});
```

**Consistent Styling:**
```javascript
// TomTom SDK - Pre-built styles
const map = new TomTomMap(
    { container: 'map' },
    { style: 'monoLight' } // TomTom style
);
```

### Use Direct MapLibre For:

**Custom Map Layers:**
```javascript
// Direct MapLibre - Custom data layers
maplibreMap.addSource('custom-data', {
    type: 'geojson',
    data: customGeoJSON
});

maplibreMap.addLayer({
    id: 'custom-layer',
    type: 'circle',
    source: 'custom-data',
    paint: { 'circle-radius': 8, 'circle-color': '#ff0000' }
});
```

**Advanced Styling:**
```javascript
// Direct MapLibre - Custom style modifications
maplibreMap.setPaintProperty('building-layer', 'fill-color', '#00ff00');
```

**Low-level Event Handling:**
```javascript
// Direct MapLibre - Detailed event control
maplibreMap.on('sourcedata', (event) => {
    if (event.sourceId === 'my-source' && event.isSourceLoaded) {
        console.log('Source loaded');
    }
});
```

## Hybrid Development Approach

The most effective approach combines both libraries:

```javascript
import { TomTomConfig } from '@cet/maps-sdk-js/core';
import { TomTomMap, PlacesModule } from '@cet/maps-sdk-js/map';
import { search } from '@cet/maps-sdk-js/services';
import { Marker, Popup } from 'maplibre-gl';

// Initialize with TomTom SDK
TomTomConfig.instance.put({ apiKey: 'YOUR_API_KEY' });
const map = new TomTomMap({ container: 'map' });

// Use TomTom services for business logic
const restaurants = await search({ query: 'restaurants' });
const placesModule = await PlacesModule.init(map);
placesModule.show(restaurants);

// Use MapLibre for custom features
const customMarker = new Marker({ color: '#ff0000' })
    .setLngLat([4.9, 52.3])
    .setPopup(new Popup().setText('Custom Location'))
    .addTo(map.mapLibreMap);

// Combine event handling
placesModule.events.on('click', (feature) => {
    console.log('TomTom place clicked:', feature.properties.name);
});

map.mapLibreMap.on('click', (event) => {
    console.log('Map clicked at:', event.lngLat);
});
```

## Benefits of This Integration

### For Developers
- **Flexibility**: Choose the right tool for each task
- **Future-proof**: Direct access to MapLibre ensures compatibility
- **Learning curve**: Familiar MapLibre APIs remain available
- **Community**: Access to both TomTom and MapLibre communities

### For Applications
- **Performance**: Leverage MapLibre's optimized rendering
- **Functionality**: Access TomTom's comprehensive service ecosystem
- **Customization**: Full control over map appearance and behavior
- **Scalability**: Build from simple to complex mapping applications

## MapLibre Compatibility

The TomTom SDK is designed to be compatible with MapLibre GL JS best practices:

```javascript
// SDK follows MapLibre patterns
const map = new TomTomMap({
    container: 'map',
    center: [lng, lat],    // MapLibre convention
    zoom: 12,              // MapLibre zoom levels
    style: mapStyle        // MapLibre style format
});

// Events follow MapLibre naming
map.mapLibreMap.on('load', () => {
    console.log('Map loaded');
});
```

## Getting Started

1. **Install both libraries**: `npm install @cet/maps-sdk-js maplibre-gl`
2. **Start with TomTom SDK**: Use SDK for core functionality
3. **Add MapLibre features**: Use direct MapLibre for custom needs
4. **Combine as needed**: Leverage the strengths of both libraries
