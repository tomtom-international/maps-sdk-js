The Map Display module allows events to be detected on the TomTom map layers. By default, these layers support the following events:

-   `"click"` - a [click](https://developer.mozilla.org/en-US/docs/Web/API/Element/click_event) event
-   `"contextmenu"` - a [contextmenu](https://developer.mozilla.org/en-US/docs/Web/API/Element/contextmenu_event) event
-   `"hover"` - a hover event gets fired when the user's pointer is moving over a layer
-   `"long-hover"` - a long-hover event gets fired when the user's pointer is hovering over a layer for a longer period of time

These events can be customized through the [`MapEventsConfig`](/javascript/maps/documentation/api-reference/modules/map_init#MapEventsConfig)
when initializing a [`TomTomMap`](/javascript/maps/documentation/api-reference/classes/map_init.TomTomMap).

> If, instead of events on the TomTom map layers, you're interested in defining events directly on the MapLibre [`Map`](https://maplibre.org/maplibre-gl-js-docs/api/map/)
contained by the [`TomTomMap`](/javascript/maps/documentation/api-reference/classes/map_init.TomTomMap), take a look at [MapLibre Events](https://maplibre.org/maplibre-gl-js-docs/api/events/).

# Listening for events

Event handlers can be registered per individual module in the Map Display module. For example, a hover event handler
on traffic incidents ([`VectorTilesTraffic`](/javascript/maps/documentation/api-reference/classes/map_traffic.VectorTilesTraffic) module)
can be registered with:

> Make sure you have an initialized [`TomTomMap`](/javascript/maps/documentation/api-reference/classes/map_init.TomTomMap) instance in your project.
Read more about initializing the map in the [Quickstart](/javascript/maps/documentation/guides/map-display/quickstart) guide.

```javascript
const traffic = await VectorTilesTraffic.init(map);
traffic.events.incidents.on("hover", eventHandler);
```

where `eventHandler` is a function of type [`HoverClickHandler`](/javascript/maps/documentation/api-reference/modules/map_shared#HoverClickHandler).

Remove existing listeners by calling the `off` method on the corresponding [`EventsModule`](/javascript/maps/documentation/api-reference/classes/map_shared.EventsModule).

```javascript
traffic.events.incidents.off("hover");
```

The `interactive` flag in the `config` for map modules can also be used to disable all interaction with a map layer,
effectively disabling all event handlers on that map layer.
