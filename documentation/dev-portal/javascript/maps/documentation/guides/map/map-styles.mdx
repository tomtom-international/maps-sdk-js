# Map Styles

Map styles in the TomTom Maps SDK define the visual appearance and data presentation of maps. This guide covers TomTom's built-in styles, style components, and customization options.

## Built-in Styles

### Basic Styles

```javascript
import { TomTomMap } from '@cet/maps-sdk-js/map';

// Light theme (default)
const map = new TomTomMap(
    { container: 'map' },
    { style: 'monoLight' }
);

// Dark theme
const map = new TomTomMap(
    { container: 'map' },
    { style: 'monoDark' }
);

// Satellite imagery with roads
const map = new TomTomMap(
    { container: 'map' },
    { style: 'hybrid' }
);

// Pure satellite imagery
const map = new TomTomMap(
    { container: 'map' },
    { style: 'satellite' }
);
```

## Dynamic Style Switching

```javascript
const map = new TomTomMap({ container: 'map' });

// Change style after map creation
map.setStyle('monoDark');
map.setStyle('hybrid');
```

## Styles with Additional Layers

```javascript
// Add traffic and POI layers
const map = new TomTomMap(
    { container: 'map' },
    {
        style: {
            type: 'published',
            id: 'monoLight',
            include: ['trafficIncidents', 'trafficFlow', 'pois']
        }
    }
);

// Add only traffic layers
const map = new TomTomMap(
    { container: 'map' },
    {
        style: {
            type: 'published',
            id: 'monoDark',
            include: ['trafficIncidents', 'trafficFlow']
        }
    }
);
```

## Available Layer Modules

```javascript
// Common layer modules you can include:
include: [
    'trafficIncidents',  // Traffic incidents
    'trafficFlow',       // Traffic flow data
    'pois',              // Points of interest
    'hillshade'          // Terrain shading
]
```

## Map Language

```javascript
const map = new TomTomMap(
    { container: 'map' },
    {
        style: 'monoLight',
        language: 'de-DE' // German labels
    }
);
```

## Style Switching with UI

```javascript
const map = new TomTomMap({ container: 'map' });

// Style selector example
document.getElementById('light-style').onclick = () => map.setStyle('monoLight');
document.getElementById('dark-style').onclick = () => map.setStyle('monoDark');
document.getElementById('hybrid-style').onclick = () => map.setStyle('hybrid');
```

For complete examples, check the `/examples` folder in the repository.
