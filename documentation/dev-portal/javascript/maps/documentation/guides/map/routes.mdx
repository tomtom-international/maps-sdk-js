# Routes Module

The Routes Module in the TomTom Maps SDK enables displaying calculated routes and waypoints on maps with comprehensive functionality for visualizing navigation routes and handling route-related interactions.

## Basic Usage

```javascript
import { TomTomMap, RoutesModule } from '@cet/maps-sdk-js/map';
import { calculateRoute } from '@cet/maps-sdk-js/services';

const map = new TomTomMap({ container: 'map' });
const routesModule = await RoutesModule.get(map);

// Calculate a route
const route = await calculateRoute({
    locations: [
        [4.9041, 52.3676], // Amsterdam
        [4.4777, 51.9244]  // Rotterdam
    ]
});

// Display the route
routesModule.addRoute({
    id: 'myRoute',
    route: route.routes[0]
});
```

## Adding Multiple Routes

```javascript
// Add multiple routes (e.g., alternatives)
route.routes.forEach((routeData, index) => {
    routesModule.addRoute({
        id: `route-${index}`,
        route: routeData
    });
});
```

## Route Styling

```javascript
routesModule.addRoute({
    id: 'myRoute',
    route: route.routes[0],
    style: {
        lineColor: '#FF0000',
        lineWidth: 5,
        lineOpacity: 0.8
    }
});
```

## Managing Routes

```javascript
// Remove a route
routesModule.removeRoute('myRoute');

// Clear all routes
routesModule.clear();

// Get all route IDs
const routeIds = routesModule.getRouteIds();

// Check if route exists
if (routesModule.hasRoute('myRoute')) {
    console.log('Route exists');
}
```

## Route with Waypoints

```javascript
import { geocode } from '@cet/maps-sdk-js/services';

// Geocode waypoints
const origin = await geocode({ query: 'Amsterdam Central Station' });
const destination = await geocode({ query: 'Rotterdam Central Station' });

const route = await calculateRoute({
    locations: [
        origin.features[0].geometry.coordinates,
        destination.features[0].geometry.coordinates
    ]
});

routesModule.addRoute({
    id: 'train-route',
    route: route.routes[0]
});
```

## Route Events

```javascript
// Listen for route clicks
map.getMaplibreMap().on('click', (event) => {
    const features = map.getMaplibreMap().queryRenderedFeatures(event.point, {
        layers: routesModule.getLayerIds()
    });

    if (features.length > 0) {
        const routeId = features[0].properties.routeId;
        console.log(`Clicked route: ${routeId}`);
    }
});
```
