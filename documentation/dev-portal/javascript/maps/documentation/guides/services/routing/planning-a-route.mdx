# Planning a Route

Route planning in the TomTom Maps SDK involves configuring parameters to calculate optimal routes that meet specific requirements, from basic route requests to advanced optimization strategies.

## Basic Route Planning

```javascript
import { calculateRoute, geocode } from '@cet/maps-sdk-js/services';

// Simple route between two cities
const route = await calculateRoute({
    locations: [
        [4.9041, 52.3676], // Amsterdam
        [2.3522, 48.8566]  // Paris
    ]
});
```

## Route with Address Geocoding

```javascript
// Using addresses instead of coordinates
const origin = await geocode({ query: 'Amsterdam Central Station' });
const destination = await geocode({ query: 'Paris Gare du Nord' });

const route = await calculateRoute({
    locations: [
        origin.features[0].geometry.coordinates,
        destination.features[0].geometry.coordinates
    ]
});
```

## Route Types

```javascript
// Fastest route (default)
const fastestRoute = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    routeType: 'fastest'
});

// Shortest route
const shortestRoute = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    routeType: 'shortest'
});

// Eco-friendly route
const ecoRoute = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    routeType: 'eco'
});
```

## Route with Traffic

```javascript
const route = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    traffic: true,
    departAt: 'now' // or specific datetime
});
```

## Route Avoidances

```javascript
const route = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    avoid: ['tollRoads', 'ferries', 'unpavedRoads']
});
```

## Alternative Routes

```javascript
const routes = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    maxAlternatives: 2 // Get up to 3 routes total (main + 2 alternatives)
});

routes.routes.forEach((route, index) => {
    console.log(`Route ${index + 1}:`);
    console.log(`Distance: ${route.summary.lengthInMeters}m`);
    console.log(`Duration: ${route.summary.travelTimeInSeconds}s`);
});
```

## Vehicle-Specific Routing

```javascript
// Electric vehicle routing
const evRoute = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    vehicleEngineType: 'electric',
    vehicleWeight: 1600, // kg
    vehicleMaxSpeed: 90, // km/h
    vehicleCommercial: false
});

// Truck routing
const truckRoute = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    vehicleEngineType: 'combustion',
    vehicleWeight: 12000, // kg
    vehicleAxleWeight: 9000, // kg
    vehicleLength: 12, // meters
    vehicleWidth: 2.5, // meters
    vehicleHeight: 4, // meters
    vehicleCommercial: true
});
```

## Route with Departure Time

```javascript
// Route for specific departure time
const futureDate = new Date();
futureDate.setHours(futureDate.getHours() + 2);

const route = await calculateRoute({
    locations: [[4.9041, 52.3676], [2.3522, 48.8566]],
    departAt: futureDate.toISOString(),
    traffic: true
});
```
