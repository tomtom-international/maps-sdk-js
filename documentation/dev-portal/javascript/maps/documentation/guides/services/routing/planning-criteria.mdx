# Route Planning Criteria Overview

Route planning criteria in the TomTom Maps SDK define how routes are optimized and calculated. This guide covers the comprehensive set of parameters and options available to customize route planning for different use cases, vehicle types, and optimization goals.

## Purpose and Functionality

Planning criteria enable you to:
- **Optimize routes** based on different objectives (time, distance, fuel efficiency)
- **Configure vehicle-specific constraints** for cars, trucks, motorcycles, and electric vehicles
- **Apply routing preferences** such as avoiding tolls, highways, or specific areas
- **Control route characteristics** including scenic routes, terrain preferences, and road types
- **Integrate real-time factors** like traffic conditions and departure times

## Core Planning Criteria Categories

### Route Optimization Types

#### Route Type Options
Different optimization strategies for different use cases:

```typescript
import { calculateRoute } from '@cet/maps-sdk-js/services';

// Fastest route - minimize travel time
const fastestRoute = await calculateRoute({
    geoInputs: [origin, destination],
    routeType: 'fastest',
    traffic: true
});

// Shortest route - minimize distance
const shortestRoute = await calculateRoute({
    geoInputs: [origin, destination],
    routeType: 'shortest'
});

// Eco-friendly route - minimize fuel consumption
const ecoRoute = await calculateRoute({
    geoInputs: [origin, destination],
    routeType: 'eco',
    constantSpeedConsumptionInLitersPerHundredKm: {
        "50": 6.5,
        "90": 7.2,
        "120": 8.5
    }
});

// Thrilling route - scenic/winding roads (motorcycles)
const thrillingRoute = await calculateRoute({
    geoInputs: [origin, destination],
    routeType: 'thrilling',
    hilliness: 'high',
    windingness: 'high'
});
```

#### Route Type Characteristics

**Fastest Route:**
- Optimizes for minimum travel time
- Considers traffic conditions when enabled
- Prefers highways and major roads
- Best for time-sensitive journeys

**Shortest Route:**
- Optimizes for minimum distance
- May use smaller roads and avoid highways
- Useful for fuel cost optimization
- Good for local deliveries

**Eco Route:**
- Optimizes for fuel efficiency
- Considers vehicle consumption patterns
- Balances distance and speed for efficiency
- Requires vehicle fuel consumption data

**Thrilling Route:**
- Optimizes for scenic and enjoyable driving
- Prefers winding and hilly roads
- Ideal for motorcycles and leisure driving
- May increase travel time and distance

### Traffic Integration

#### Traffic-Aware Routing

```typescript
// Current traffic conditions
const currentTrafficRoute = await calculateRoute({
    geoInputs: [origin, destination],
    traffic: true,
    departureTime: new Date() // Use current time
});

// Future traffic prediction
const futureTrafficRoute = await calculateRoute({
    geoInputs: [origin, destination],
    traffic: true,

});

// Historical traffic patterns
const historicalRoute = await calculateRoute({
    geoInputs: [origin, destination],
    traffic: false, // Use historical patterns instead
    routeType: 'fastest'
});
```

## Route Avoidance and Preferences

### Basic Avoidances

```typescript
const avoidanceRoute = await calculateRoute({
    geoInputs: [origin, destination],

    // Road type avoidances
    avoidTolls: true,           // Avoid toll roads
    avoidMotorways: false,      // Allow highways
    avoidFerries: true,         // Avoid ferry crossings
    avoidUnpaved: true,         // Stick to paved roads
    avoidCarpool: true,         // Avoid HOV/carpool lanes
    avoidBorderCrossings: false, // Allow international borders

    // Traffic avoidances

    routeType: 'fastest'
});
```

### Geographic Area Avoidances

```typescript
// Avoid specific geographic areas

    geoInputs: [origin, destination],

    // Area to avoid

        boundingBox: {
            topLeftPoint: [4.8, 52.4],     // Northwest corner
            bottomRightPoint: [4.9, 52.3]  // Southeast corner
        }
    },

    routeType: 'fastest'

    geoInputs: [origin, destination],

        boundingBoxes: [
            {
                topLeftPoint: [4.8, 52.4],
                bottomRightPoint: [4.9, 52.3]
            },
            {
                topLeftPoint: [4.7, 52.2],
                bottomRightPoint: [4.8, 52.1]
            }
        ]
    },

    routeType: 'fastest'
});
```

## Criteria Validation and Best Practices

### Parameter Validation

```typescript
function validateRoutingCriteria(params: any): { isValid: boolean; errors: string[] } {
    const errors: string[] = [];

    // Vehicle weight limits
    if (params.vehicleWeight && params.vehicleWeight > 60000) {
        errors.push('Vehicle weight exceeds maximum limit of 60,000 kg');
    }

    // Dimension checks
    if (params.vehicleHeight && params.vehicleHeight > 4.5) {
        errors.push('Vehicle height exceeds typical bridge clearance');
    }

    // Electric vehicle parameters
    if (params.vehicleEngineType === 'electric') {
        if (!params.maxChargeInkWh) {
            errors.push('Electric vehicles require maxChargeInkWh parameter');
        }

        if (params.currentChargeInkWh > params.maxChargeInkWh) {
            errors.push('Current charge cannot exceed maximum battery capacity');
        }
    }

    // Alternative limits
    if (params.maxAlternatives && params.maxAlternatives > 5) {
        errors.push('Maximum 5 alternative routes allowed');
    }

    return {
        isValid: errors.length === 0,
        errors
    };
}
```

## Complete Criteria Example

```typescript
import { TomTomConfig } from '@cet/maps-sdk-js/core';
import { calculateRoute, geocode } from '@cet/maps-sdk-js/services';

TomTomConfig.instance.put({ apiKey: 'YOUR_API_KEY' });

async function comprehensiveRoutingCriteria() {
    try {
        // Get waypoints
        const origin = (await geocode({ query: 'Amsterdam, Netherlands' })).features[0];
        const destination = (await geocode({ query: 'Munich, Germany' })).features[0];

        // Define comprehensive routing criteria
        const routingCriteria = {
            geoInputs: [origin, destination],

            // Route optimization
            routeType: 'eco',
            traffic: true,
            departureTime: new Date(),

            // Vehicle specifications
            vehicleEngineType: 'combustion',
            constantSpeedConsumptionInLitersPerHundredKm: {
                "50": 6.5,
                "90": 7.2,
                "120": 8.5
            },
            currentFuelInLiters: 50,
            auxiliaryPowerInLitersPerHour: 0.2,

            // Route preferences
            avoidTolls: false,              // Allow tolls for long distance
            avoidMotorways: false,          // Prefer highways
            avoidFerries: true,             // Avoid ferry crossings

            // Terrain preferences
            hilliness: 'normal',
            windingness: 'low',             // Prefer direct routes

            // Alternative routes
            maxAlternatives: 1,
            alternativeType: 'betterRoute',

            // Output configuration
            instructionsType: 'text',
            language: 'en-GB',
            computeTravelTimeFor: 'all',

            // Report configuration
            sectionType: 'traffic',
            report: 'effectiveSettings'
        };

        // Validate criteria
        const validation = validateRoutingCriteria(routingCriteria);
        if (!validation.isValid) {
            throw new Error('Invalid routing criteria: ' + validation.errors.join(', '));
        }

        // Calculate route
        const routeResult = await calculateRoute(routingCriteria);

        console.log('Route calculated with comprehensive criteria:');
        const route = routeResult.routes[0];
        console.log(`Distance: ${(route.summary.lengthInMeters / 1000).toFixed(1)} km`);
        console.log(`Time: ${Math.round(route.summary.travelTimeInSeconds / 60)} min`);
        console.log(`Fuel: ${route.summary.fuelConsumptionInLiters?.toFixed(2) || 'N/A'} L`);

        return routeResult;

    } catch (error) {
        console.error('Route planning with criteria failed:', error);
        throw error;
    }
}

// Execute comprehensive routing
comprehensiveRoutingCriteria();
```
