The camera position controls how the map is displayed. Camera position is handled by the MapLibre
[`Map`](https://maplibre.org/maplibre-gl-js-docs/api/map/) contained by the [`GOSDKMap`](/maps-sdk-js/javascript/maps/documentation/api-reference/classes/map.GOSDKMap).
The MapLibre [`Map`](https://maplibre.org/maplibre-gl-js-docs/api/map/) has options such as `center`, `bearing`, `pitch` and `zoom` for changing the default camera position.
It also exposes members for animating a camera move, such as `easeTo(...)`, `flyTo(...)` and `jumpTo(...)`.
For more information, see the [`Map`](https://maplibre.org/maplibre-gl-js-docs/api/map/) in the MapLibre API reference and the [MapLibre Camera examples](https://maplibre.org/maplibre-gl-js-docs/example/?topic=Camera) .
