The Map Display module allows you to show the [`GOSDKMap`](/maps-sdk-js/javascript/maps/documentation/api-reference/classes/map.GOSDKMap) in your application.
The [`GOSDKMap`](/maps-sdk-js/javascript/maps/documentation/api-reference/classes/map.GOSDKMap) is built upon and exposes a MapLibre [`Map`](https://maplibre.org/maplibre-gl-js-docs/api/map/).

To meet your use case requirements, you can apply initial settings during initialization of the map using [`MapLibreOptions`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#MapLibreOptions) and [`GOSDKMapParams`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#GOSDKMapParams).

{<a style={{display: 'block', margin: '0', padding: '0'}} id="_initializing_the_map"></a>}

# Initializing the map

The [`GOSDKMap`](/maps-sdk-js/javascript/maps/documentation/api-reference/classes/map.GOSDKMap) is initialized using two parameters:
[`MapLibreOptions`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#MapLibreOptions) and [`GOSDKMapParams`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#GOSDKMapParams).
The [`MapLibreOptions`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#MapLibreOptions) parameter is based on the options for a MapLibre [`Map`](https://maplibre.org/maplibre-gl-js-docs/api/map/)
and is used to initialize the MapLibre [`Map`](https://maplibre.org/maplibre-gl-js-docs/api/map/) contained by the [`GOSDKMap`](/maps-sdk-js/javascript/maps/documentation/api-reference/classes/map.GOSDKMap).
The [`GOSDKMapParams`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#GOSDKMapParams) parameter is an optional parameter and is used to customize the map styles and map event configuration.

To initialize a [`GOSDKMap`](/maps-sdk-js/javascript/maps/documentation/api-reference/classes/map.GOSDKMap) in your project, first add your API key to your project:

```javascript
 GOSDKConfig.instance.put({
     // (Set your own API key when working in your own environment)
     apiKey: process.env.API_KEY
 });
```

Then, to initialize a basic [`GOSDKMap`](/maps-sdk-js/javascript/maps/documentation/api-reference/classes/map.GOSDKMap) instance without [`GOSDKMapParams`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#GOSDKMapParams):

<Code>

```typescript
import { GOSDKMap } from "@anw/go-sdk-js/map";

const goSDKMap = new GOSDKMap(
    {
        // Container element defined in the corresponding HTML file
        container: document.getElementById("map") as HTMLElement,
        zoom: 14,
        minZoom: 2,
        center: [-0.12621, 51.50394]
    }
);
```

```javascript
import { GOSDKMap } from "@anw/go-sdk-js/map";

const goSDKMap = new GOSDKMap(
    {
        // Container element defined in the corresponding HTML file
        container: document.getElementById("map"),
        zoom: 14,
        minZoom: 2,
        center: [-0.12621, 51.50394]
    }
);
```

</Code>

The `container` property of the [`MapLibreOptions`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#MapLibreOptions) is the only required parameter for initializing a basic map.
More options for changing the initial state of the map and map control, like `zoom` and `center`, can be found in [`MapLibreOptions`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#MapLibreOptions).

Adding options for styling the map and event configuration is done through the [`GOSDKMapParams`](/maps-sdk-js/javascript/maps/documentation/api-reference/modules/map#GOSDKMapParams).
